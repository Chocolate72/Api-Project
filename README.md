# Weather API Demo

This project demonstrates how to connect to a public API using JavaScript.

## API Used
OpenWeatherMap API  
https://openweathermap.org/current

## Features
- User inputs a city name
- Fetches live weather data
- Displays temperature, humidity, wind, sunrise & sunset times
- Uses JavaScript fetch()
- Processes JSON output

## Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeatherMap API

## How To Run
1. Open index.html in browser
2. Enter city name
3. Click "Get Weather"

## GitHub Requirement
This project must be pushed to GitHub with multiple commits showing progress.
